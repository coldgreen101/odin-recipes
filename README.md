# Odin Recipes

A basic recipe website built as part of [The Odin Project](https://www.theodinproject.com/) Foundations course.

## About

This project practices core HTML fundamentals: page structure, links, lists, images, and multi-page navigation. It contains an index page that links to individual recipe pages, each with a description, ingredients list, and step-by-step instructions.

## Recipes

- [Lasagna](recipes/lasagna.html)
- [French Onion Soup](recipes/french-onion-soup.html)
- [Banana Bread](recipes/banana-bread.html)
